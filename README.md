## Descripción de la práctica

Fabio Montes. 15-12-2025.

Esta práctica introduce el uso básico de **Git y GitHub** trabajando con archivos de texto en formato **Markdown (.md)**. El objetivo principal es aprender a gestionar un repositorio mediante **commits**, **ramas (branches)** y **Pull Requests**, así como comprender el flujo de trabajo habitual en proyectos colaborativos.  
Durante la práctica se crearán distintas ramas, se editarán archivos, se generarán conflictos de fusión y se resolverán mediante Pull Requests, permitiendo además visualizar el histórico de commits y ramas.
